Two Sum :

Thought process :
We can use a hashmap to store the elements of the array and their indices.
We can then iterate through the array and check if the complement (target - current element) exists in the hashmap.
If it does, we return the indices of the current element and the complement.

class Solution {
    public int[] twoSum(int[] nums, int target) {
        HashMap<Integer,Integer> hmap = new HashMap<>();
        for(int i = 0;i<nums.length;i++){
            if(hmap.containsKey(target-nums[i])){
                return new int[] {i,hmap.get(target-nums[i])};
            }
            hmap.put(nums[i],i);
        }
        return new int[0];
    }
}

Valid Anagaram :

Thought process :
We can use a hashmap to store the frequency of each character in the first string.
We can then iterate through the second string and decrement the frequency of each character in the hashmap.
Finally, we can check if all the values in the hashmap are zero, which means that the two strings are anagrams.

class Solution {
    public boolean isAnagram(String s, String t) {
        HashMap<Character,Integer> hmap = new HashMap<>();
        if(s.length() != t.length()){
            return false;
        }
        for(int i = 0;i<s.length();i++){
            char c = s.charAt(i);
            hmap.put(c,hmap.getOrDefault(c,0)+1);
        }
        for(int i = 0;i<t.length();i++){
            char c = t.charAt(i);
            hmap.put(c,hmap.getOrDefault(c,0)-1);
        }
        for( int i : hmap.values()){
            if(i!=0){
                return false;
            }
        }
        return true;
    }
}

Contains Duplicate :

Thought process : We can use a hashset to store the elements of the array.
We can then iterate through the array and check if the current element already exists in the hashset.
If it does, we return true. If it does not, we add it to the hashset.
If we finish iterating through the array without finding any duplicates, we return false.

class Solution {
    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> hset = new HashSet<>();

        for(int i : nums){
            if (hset.contains(i)){
                return true;
            }
            hset.add(i);
        }
        return false;
    }
}

Best Time to Buy and Sell Stock :

Thought process : 
We can use two pointers to keep track of the minimum price and the maximum profit.
We can iterate through the array and update the minimum price and the maximum profit at each step.

class Solution {
    public int maxProfit(int[] prices) {
        int min = prices[0];
        int maxP = 0;

        for(int i  = 0;i<prices.length;i++){
            int currP = prices[i]-min;

            maxP = Math.max(currP,maxP);
            if(prices[i]<min){
                min = prices[i];
            }
        }
        return maxP;
    }
}

Valid Palindrome :

Thought process : We can use two pointers to check if the string is a palindrome.
We can ignore non-alphanumeric characters and convert all characters to lowercase.
Character.isLetterOrDigit() can be used to check if a character is alphanumeric.
Character.toLowerCase() can be used to convert a character to lowercase.

class Solution {
    public boolean isPalindrome(String s) {
        if(s.length()==0)
            return true;
        int l = 0;
        int r = s.length()-1;

        while(l<r){
            char lc = s.charAt(l);
            char rc = s.charAt(r);
            if(!Character.isLetterOrDigit(lc)){
                l++;
                continue;
            }
            if(!Character.isLetterOrDigit(rc)){
                r--;
                continue;
            }
            if(Character.toLowerCase(lc)  != Character.toLowerCase(rc)){
                        return false;
            }
            l++;
            r--;
        }
        return true;
    }
}

Two Sum II - Input Array Is Sorted :

Thought process : We can use two pointers to find the two numbers that add up to the target.
We can start one pointer at the beginning of the array and the other pointer at the end of the array.
We can then check if the sum of the two numbers is equal to the target.
If it is, we return the indices of the two numbers.
If the sum is less than the target, we can move the left pointer to the right to increase the sum.
If the sum is greater than the target, we can move the right pointer to the left to decrease the sum.

class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int l = 0;
        int r = numbers.length-1;

        while( l < r ){
            if(numbers[l]+numbers[r]<target){
                l++;
            }
            else if(numbers[l]+numbers[r]>target){
                r--;
            }
            else{
                return new int[] {l+1,r+1};
            }
        }
        return new int[] {};
    }
}

Three Sum :

Thought process : We can sort the array and then use two pointers to find the three numbers that add up to zero.
We can iterate through the array and for each element, we can use two pointers to find the other two numbers that add up to the negative of the current element.
We can then add the three numbers to the result list if they add up to zero and skip duplicates.
Skip duplicates by checking if the current element is the same as the previous element. If it is, we can continue to the next iteration.
Dont forget to check duplicates for the left and right pointers as well.

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);

        for(int i = 0;i<nums.length-2;i++){
            if(i>0 && nums[i] == nums[i-1]){
                continue;
            }
            int l = i+1;
            int r = nums.length-1;

            while(l<r){
                int sum = nums[i]+nums[l]+nums[r];
                if(sum<0){
                    l++;
                }
                else if(sum>0){
                    r--;
                }
                else{
                    res.add(Arrays.asList(nums[i],nums[l],nums[r]));
                    while(l<r && nums[l] == nums[l+1]){
                        l++;
                    }
                    while(l<r && nums[r] == nums[r-1]){
                        r--;
                    }
                    l++;
                    r--;
                }
            }
        }
        return res;
    }
}

Container With Most Water :

Thought process : We can use two pointers to find the maximum area of water that can be contained.
We can start one pointer at the beginning of the array and the other pointer at the end of the array.
We can then calculate the area of water that can be contained between the two pointers and update the maximum area if the current area is greater than the maximum area.
We can then move the pointer that has the smaller height to try to find a larger area.

class Solution {
    public int maxArea(int[] height) {
        int l = 0;
        int r = height.length-1;
        int maxA = 0;

        while(l<r){
            int currA = Math.min(height[l],height[r])*(r-l);
            maxA = Math.max(currA,maxA);
            if(height[l]<height[r]){
                l++;
            }
            else{
                r--;
            }
        }
        return maxA;
    }
}

Longest Substring Without Repeating Characters:

Thought process : We use two pointers to keep track of the current substring and a hashset to store the characters in the current substring.
We can start both pointers at the beginning of the string and iterate through the string with the right pointer.
If the character at the right pointer is not in the hashset, we can add it to the hashset and update the maximum length of the substring.
If the character at the right pointer is in the hashset, we can remove the character at the left pointer from the hashset and move the left pointer to the right until the character at the right pointer is not in the hashset.
We can then add the character at the right pointer to the hashset and update the maximum length of the substring.

class Solution {
    public int lengthOfLongestSubstring(String s) {
        HashSet<Character> hset = new HashSet<>();
        int l = 0;
        int maxL = 0;

        for(int r = 0;r<s.length();r++){
            char c = s.charAt(r);
            while(hset.contains(c)){
                hset.remove(s.charAt(l));
                l++;
            }
            hset.add(c);
            maxL = Math.max(maxL,r-l+1);
        }
        return maxL;
    }
}

Longest Repeating Character Replacement

Thought process : 



class Solution {
    public int characterReplacement(String s, int k) {
        HashMap<Character,Integer> hmap = new HashMap<>();
        int l = 0;
        int r = 0;
        int maxFreq = 0;
        int maxLen = 0;
        while(r < s.length()){
            hmap.put(s.charAt(r),hmap.getOrDefault(s.charAt(r),0)+1);
            if(hmap.get(s.charAt(r))>maxFreq){
                maxFreq = hmap.get(s.charAt(r));
            }
            if((r - l + 1) - maxFreq > k){
                hmap.put(s.charAt(l),hmap.getOrDefault(s.charAt(l),0)-1);
                l++;
            }

            maxLen = Math.max(maxLen, r - l + 1);
            r++;
        }
        return maxLen;
    }
}
